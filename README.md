## What This Is

This project is a VR/AR technology portfolio review of the [Bigscreen Beyond 2](https://www.bigscreenvr.com/), a tethered PC VR headset released in March 2025 by Bigscreen Inc. The write-up covers all required portfolio components for a 10-minute class presentation.

---

## Portfolio Topics Covered

### 1. Purpose

The Bigscreen Beyond 2 is a tethered PC VR headset built for maximum visual quality in the smallest, lightest possible form factor. It is the premium gateway into the SteamVR ecosystem and Bigscreen's own social cinema app, not a standalone device, but a pure display peripheral for a gaming PC.

### 2. Target Users

- **PC VR Enthusiasts** — SteamVR users who want top-tier visuals without headset bulk
- **Movie & Media Fans** — Bigscreen social cinema users watching films with friends in VR
- **Sim Racers & Pilots** — Cockpit-setup users with steering wheels and HOTAS rigs
- **VRChat / Social VR** — Avatar-based social communities; eye tracking on the 2e enables lip sync and gaze expressions
- **VR Productivity** — Virtual multi-monitor desktop streaming via Virtual Desktop
- **NOT** for beginners, children, or anyone without an existing SteamVR setup

### 3. Features

- Bigscreen Social Cinema App — up to 16 people in shared virtual movie theaters
- Full SteamVR gaming library — Half-Life: Alyx, Beat Saber, Flight Simulator VR, iRacing, and more
- Virtual Desktop — stream your PC as floating virtual monitors in any environment
- Eye Tracking (Beyond 2e variant) — 90 Hz, 400x400 per eye, adds only 1.05g
- Room-scale 6DoF tracking via SteamVR base stations
- Full-body tracking support with HTC Vive and Tundra Trackers

### 4. Technology

| Spec | Value |
|------|-------|
| Display | 2x 1-inch Micro-OLED |
| Resolution per Eye | 2560 x 2560 px |
| Lenses | Pancake Optics |
| Field of View | 116 degrees diagonal |
| Refresh Rate | 75 Hz / 90 Hz |
| Weight | 107g (headset only) |
| Tracking | SteamVR 1.0 and 2.0 |
| IPD Range | 48 – 75 mm (adjustable) |
| Price | $1,019 USD |

Tethering to a PC is a deliberate design choice, it enables micro-OLED displays, sub-millisecond tracking, and eliminates battery constraints. The trade-off is that the headset must always be near a gaming PC.

### 5. Visual Experience

- Micro-OLED panels deliver perfect blacks and vibrant colors with no backlight bleed
- Pancake lenses provide edge-to-edge clarity and a large sweet spot compared to the fresnel lenses in the original Beyond
- Realism is a stated goal, OLED self-emissive pixels and high pixel density are specifically chosen to maximize immersion
- Pancake lenses reduce light transmission by approximately 50%, so bright outdoor VR scenes appear dimmer than on LCD headsets

### 6. Interaction

- SteamVR controllers (Valve Index / HTC Vive) — industry-leading 6DoF accuracy, near-zero latency
- Sim peripherals (wheels, HOTAS) — controllers are entirely optional for cockpit-style setups
- Bigscreen laser pointer — simple virtual pointer for navigating the cinema app
- Gaze interaction (Beyond 2e) — eye tracking enables foveated rendering and social VR expressions
- No hand tracking — no built-in cameras for gesture recognition
- No passthrough or mixed reality support

### 7. Limitations

| Limitation | Details |
|-----------|---------|
| No standalone mode | Requires a tethered, VR-ready gaming PC |
| No passthrough / mixed reality | No cameras — fully immersive VR only |
| High total cost | $1,019 headset + base stations + controllers + PC = $2,000+ total |
| No bundled controllers | Must purchase SteamVR controllers separately |
| Reduced brightness | Pancake lenses cut approximately 50% of light vs. fresnel |
| Complex setup | 2+ base stations, 5m cable, and SteamVR configuration required |

### 8. What Developers Should Do Next

1. Wireless streaming — Wi-Fi 7 or UWB to eliminate the tether cable
2. Higher-brightness OLED — offset pancake lens light loss as panel technology matures
3. Foveated rendering — full eye-tracked dynamic foveation to cut GPU load by 40–60%
4. Hand tracking — add inside-out cameras for gesture-based interaction
5. 120 Hz+ refresh rate — smoother fast-motion gaming
6. Expand Bigscreen app — more studio partnerships, live event streaming, mixed-reality desktop integration
